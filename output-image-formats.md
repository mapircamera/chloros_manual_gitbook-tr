# Çıktı Görüntü Biçimleri

### TIFF (16 bit)

16 bit (piksel dijital numarası 0 - 65535) TIFF (.tif) biçiminde çıktı.

### TIFF (32 bit, Yüzde %)

32 bit (piksel float 0,0 - 1,0) TIFF (.tif) formatında çıktı. Piksel değerleri yansıma yüzdelerini temsil eder; burada 0,0 = %0 ve 1,0 = %100&#x27;dür.

### **JPG (8 bit)**

8 bit (piksel dijital numarası 0 -255) JPG (.jpg) formatında çıktı.

### **PNG (8 bit)**

8 bit (piksel dijital numarası 0 - 255) PNG (.png) formatında çıktı. Web görüntüleme ve görselleştirme için uygun kayıpsız sıkıştırma.
