# Çıktı Görüntü Formatları

### TIFF (16 bit)

16 bit (piksel dijital numarası 0 - 65535) TIFF (.tif) formatında çıkış.

### TIFF (32 bit, Yüzde %)

32 bit (pixel float 0,0 - 1,0) TIFF (.tif) formatında çıktı. Piksel değerleri, 0,0 = %0 ve 1,0 = %100 olmak üzere yansıma yüzdelerini temsil eder.

### **JPG (8 bit)**

8 bit (piksel dijital sayı 0 -255) JPG (.jpg) formatında çıktı.

### **PNG (8 bit)**

8 bit (piksel dijital sayısı 0 - 255) PNG (.png) formatında çıktı. Web görüntüleme ve görselleştirme için uygun kayıpsız sıkıştırma.
